Class ActiveWarehouse::View::SqlView
In: vendor/plugins/activewarehouse/lib/active_warehouse/view/sql_view.rb
Parent: Object

A view that can figure out the SQL used by a report, rather than the results generated by executing that SQL. Useful if you want to allow users to copy the SQL and manipulate it in another more general SQL interface to your datawarehouse.
