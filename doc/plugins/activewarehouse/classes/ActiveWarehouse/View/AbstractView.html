Module ActiveWarehouse::View::AbstractView
In: vendor/plugins/activewarehouse/lib/active_warehouse/view/abstract_view.rb

Abstract version of a view that displays data generated by a ActiveWarehouse::Report::AbstractReport. Takes care of basic set up, but expects the classes that it is mixed into to implement the code that turns the report results into HTML.

Public Class methods

new(report, params, options = {})

Create a new View that displays data generated by the report. The params are used to figure out the row_dimension and column_dimension (TODO: these could probably be delegated to the report). The options are unused, but may be useful for concrete implementations of AbstractView for configuring how the report should be displayed, for example directives for showing totals, or sortable columns.

Public Instance methods

parse(html_params, stage_key, param_prefix)

Helper for constructing part of the data needed to create a ActiveWarehouse::Report::Dimension. Extracts the stage and ancestors of the hierarchy into a hash. This could probably be moved into ActiveWarehouse::Report::AbstractReport, and further cleaned up so it was less dependent on the request params.
